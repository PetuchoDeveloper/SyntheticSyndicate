# 🏴 Synthetic Syndicate

**Synthetic Syndicate** is a high-performance, heavily curated and taste-tuned, autonomous AI agent workforce designed for end-to-end full-stack development and adversarial security auditing using claude code. 

Guided by the principle of **"Decide, Delegate, Verify,"** the Syndicate operates as a tiered organization of specialized intelligence units, ensuring that every line of code is not only functional but stress-tested against the highest standards of security and UX.

## 🏗️ Workforce (Coding Line)
The **Coding Line** is the production engine of the Syndicate, managed by a strict hierarchy that prioritizes architectural integrity and modularity.

| Agent | Persona | Strategic Role |
|---|---|---|
| **Orchestrator** | 🏭 Manager | Coordinates all operations; delegates and verifies ALL work. |
| **Scheduler** | 📅 Planner | Refines the backlog and plans prioritized, sequential sprints. |
| **Backend Architect** | ⚙️ Builder | Owns the API, models, and server-side business logic. |
| **Frontend Developer** | 🎨 Builder | Crafts pixel-perfect React components and smooth UI state. |
| **Mobile App Builder** | 📱 Builder | Specializes in native and cross-platform mobile delivery. |
| **Git Specialist** | 🔀 Operative | Automates the full git lifecycle (branching, commits, merges). |
| **Documenter** | 📝 Chronicler | Maintains real-time documentation (`CLAUDE.md`) for every module. |

## 🛡️ The Hats (QA Pipeline)
The **QA Pipeline** is an adversarial strike team that subjects code to rigorous stress-testing before it reaches production.

*   **⚪ Security White Hat**: The primary auditor. Thoroughly scans for vulnerabilities and best practice violations.
*   **⚫ Security Black Hat**: The adversarial skeptic. Challenges the White Hat's findings to eliminate noise and uncover overlooked exploits.
*   **🔘 Grey Hat Orchestrator**: The final arbiter. Synthesizes conflicting reports into a single, actionable execution plan.
*   **🎭 UX Hats (White/Black)**: A parallel adversarial pair that ensures both usability standards (White) and edge-case "chaos" testing (Black).

## 🚀 Capabilities & Focus
*   **Pure Delegation**: Managed by an Orchestrator that never writes code but ensures the best specialist handles every task.
*   **Adversarial Integrity**: A recursive feedback loop between offensive and defensive agents ensures zero-tolerance for weak code.
*   **Context-Rich Automation**: Agents are launched with focused, surgical context (~4000 token budget) based on real-time codebase analysis.
*   **Immutable Standards**: Automated linting hooks, "Vibe Coding" summaries, and continuous documentation sync, agents write code, documment it for the next ones ensuring memory isnt lost, every task has transparence, you can verify the work of each agent in the task reports and decide wether it is good or not.
*   **QA coding loop**: You can even use the report generated by the QA pipeline to delegate fixes for the agents, creating clean code, and a safe environment for development. or you can use it yourself, to avoid the burden of checking manually for issues.
